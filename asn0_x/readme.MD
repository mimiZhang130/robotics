File Name: asn0_stream_sonar.py
Corresponding Step in Lab: 3
Description: streams sonar sensor output to terminal for 1 second
Run instructions:
    '''sudo python3 asn0_stream_sonar.py'''

File Name: asn0_collect_readings.py
Corresponding Step in Lab: 4
Description: gathers 10 readings from sensor for each of 3 distances: 10, 20, and 30 cm leaving a 10 second sleep time between each distance reading for the user to adjust in time
Run instructions:
    '''sudo python3 asn0_collect_readings.py'''
